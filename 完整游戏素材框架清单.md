# 完整游戏素材框架清单

本文档用于整理《余光未熄》完整游戏级素材需求。素材分为角色、地图、玩法物件、光影特效、UI、图标、音效、背景音乐和字体九大类。

## 1. 素材制作原则

- 优先保证游戏可运行、可演示、可替换。
- 角色动画使用精灵表，不建议制作成几百张散图。
- 光影特效可以先用 Raylib 几何图形实现，后续再替换为图片或粒子素材。
- 地图素材优先做可复用 tile 和房间背景，避免每个房间完全重画。
- UI 素材优先保证主菜单、任务面板、状态条和胜负结算完整。
- 音效优先覆盖核心反馈：点击、移动、点灯、任务完成、胜利、失败。

### 2.5 角色外形设计规范

整体美术风格统一为 Q 版二头身像素风动漫 RPG 角色。角色头部接近全身一半，身体短小，四肢圆润压缩，站姿为正面略微三分之二。像素表现需要有明显方块像素边缘、粗黑色或深色外轮廓、块状阴影和有限色阶，不使用柔和渐变。

#### 2.5.1 守灯人

角色定位：

- 人类阵营核心辅助角色。
- 主要负责点灯、守护、净化。
- 视觉关键词是灯杖、提灯、暖光、短斗篷。

整体轮廓：

- 二头身比例，大头小身。
- 头部接近全身一半，身体短小。
- 站姿稳定，正面略微三分之二。
- 角色剪影核心是提灯或灯杖。

脸部设计：

- 浅肤色圆脸。
- 大眼睛，眼神冷静、温和。
- 鼻子省略，小嘴。
- 表情可爱但有责任感。

发型设计：

- 暖棕色或浅金棕色短发。
- 发束大块分明，前额有几撮厚重刘海。
- 发尾略微外翘。
- 使用深棕描边，亮部用浅金色像素块。

服装设计：

- 主色为暖黄色、琥珀色、深青色。
- 上身是短款守灯外套。
- 肩部有小护肩，像轻量神殿守卫装备。
- 胸前有灯形徽记。
- 腰部有深色腰带，挂小灯油瓶或符牌。
- 下身是短裙甲或短护袍。
- 靴子为深棕短靴，边缘带金属扣。

武器和职业符号：

- 使用大号灯杖或提灯。
- 灯杖长度接近角色全身高度。
- 灯芯发出暖黄色光。
- 灯框为铜金色，轮廓夸张。
- 可以有小型光环或十字星像素高光。

像素表现重点：

- 外轮廓使用深棕或接近黑色。
- 灯光区域使用黄色、橙色、浅白三级色块。
- 身体细节不宜过多，重点突出灯、斗篷和胸前徽记。

#### 2.5.2 工程师

角色定位：

- 人类阵营机关修复角色。
- 主要负责供能、电闸、机关任务。
- 视觉关键词是护目镜、工具带、大扳手、供能装置。

整体轮廓：

- 二头身比例，身体比守灯人略厚实。
- 站姿更稳，双脚可以稍微分开。
- 角色剪影核心是工具包、大扳手和护目镜。

脸部设计：

- 浅肤色圆脸。
- 大眼睛，表情专注、可靠。
- 小嘴略微抿起。
- 整体气质聪明、能动手、可信赖。

发型设计：

- 深棕色或黑褐色短发。
- 头发略乱，体现经常工作和修理设备。
- 额头可佩戴护目镜。
- 护目镜镜片使用浅蓝或浅灰像素高光。

服装设计：

- 主色为橙色、土黄色、深灰色。
- 上身是短款工程夹克。
- 内搭深色工作服。
- 肩部有轻型护肩或金属护片。
- 手腕有护腕或机械手套。
- 腰部工具带明显，挂小扳手、螺丝刀、线圈。
- 下身是短裤或短版工装裤。
- 靴子厚实，表现工作靴质感。

武器和职业符号：

- 使用夸张大扳手，长度接近角色半身到全身。
- 可背一个小型供能电池箱。
- 扳手边缘有金属高光。
- 可加入蓝色电流小像素点作为供能符号。

像素表现重点：

- 工程师剪影要更硬朗，多使用直线和金属块。
- 工具带、护目镜、扳手是主要识别点。
- 色彩控制为橙色主体、深灰金属、蓝色电光点缀。

#### 2.5.3 观测员

角色定位：

- 人类阵营侦查和提示角色。
- 主要负责观察、预警、远程辅助。
- 视觉关键词是观测镜、星盘、记录板、轻斗篷。

整体轮廓：

- 二头身比例，身体最轻巧。
- 站姿偏灵活，略微侧身。
- 角色剪影核心是观测镜、星盘或望远镜。

脸部设计：

- 浅肤色圆脸。
- 大眼睛，比其他角色更明亮。
- 表情冷静、敏锐。
- 小嘴，整体气质安静。

发型设计：

- 深蓝黑、银灰或紫灰色头发。
- 发型可以稍长，侧边有一撮明显大块发束。
- 后发略蓬松，形成轻盈轮廓。
- 高光使用浅蓝灰或淡紫色块。

服装设计：

- 主色为蓝色、白色、紫色。
- 上身是短斗篷或观测披肩。
- 胸前有星形、眼睛或镜片符号。
- 肩部轻护肩，形状更圆。
- 腰带上挂小记录本、符文片或观测仪。
- 下身可以是短袍、短裤或轻便裙甲。
- 靴子较细，强调灵活。

武器和职业符号：

- 使用大号观测镜、短法杖或星盘。
- 武器不强调攻击性，更像侦查仪器。
- 星盘大小接近角色半身。
- 可以带小型悬浮光点，表示扫描或观察。

像素表现重点：

- 轮廓比守灯人和工程师更轻。
- 颜色偏冷，蓝紫为主。
- 星盘、镜片、高亮眼睛是主要识别点。
- 可加入少量发光像素，但不要过度华丽。

#### 2.5.4 影角色

角色定位：

- 对抗阵营角色。
- 主要表现黑影、黑雾、同化、传送。
- 视觉关键词是黑雾身体、发光眼睛、雾爪、粒子拖尾。

整体轮廓：

- 二头身或接近二头身，但身体边界不完全固定。
- 看起来像小型黑雾人形。
- 头部偏大，身体向下逐渐化成雾。
- 没有清晰双腿，底部可以是飘散黑雾。
- 角色剪影核心是黑雾身体、发光眼睛和粒子拖尾。

脸部设计：

- 没有正常人脸。
- 只有一双发光眼睛。
- 眼睛可以是紫色、冷白色或暗红色。
- 嘴可有可无，如果有，只做一条细小裂缝。
- 表情冷、空洞、有压迫感。

头部和黑雾结构：

- 头部像黑色影团。
- 边缘不是完全圆形，而是雾状尖刺和破碎像素。
- 上方可以有两撮像角或烟雾尖端的形状。
- 内部使用深黑、紫黑、灰黑三档色块。

身体设计：

- 上半身隐约有人形。
- 有短小手臂，但像雾气凝成。
- 手爪可以稍尖，但保持 Q 版。
- 下半身逐渐消散，没有明确脚部。
- 底部有黑雾拖尾和小粒子。

特殊效果：

- 身边漂浮黑紫色粒子。
- 粒子大小使用 1 到 3 像素点表现。
- 边缘使用淡紫或暗蓝描边，避免纯黑看不清。
- 虚弱状态时身体变淡，边缘破碎，眼睛变暗。
- 显形状态时轮廓更清晰，眼睛更亮。

职业符号和能力符号：

- 不需要实体武器。
- 主要符号是黑雾、裂纹、粒子、传送残影。
- 同化攻击时可以伸出雾爪。
- 传送时身体可以被拉成竖向烟雾。

像素表现重点：

- 不要画成普通黑衣人，要表现为黑影和雾气构成的角色。
- 轮廓必须清楚，建议用深紫描边。
- 内部加入少量紫色高光，表现能量流动。
- 粒子特效围绕身体，但不要太分散，避免动画时难管理。

#### 2.5.5 四个角色主题区分

| 角色 | 主色 | 剪影关键词 | 职业符号 |
|---|---|---|---|
| 守灯人 | 琥珀黄 + 深青 | 灯杖、短斗篷、暖光 | 提灯 / 灯杖 |
| 工程师 | 橙色 + 深灰 | 护目镜、工具带、厚靴 | 大扳手 / 电池箱 |
| 观测员 | 蓝紫 + 白 | 星盘、披肩、轻装 | 观测镜 / 星盘 |
| 影角色 | 黑紫 + 冷光 | 黑雾、人形残影、粒子 | 发光眼睛 / 雾爪 |

制作时建议守灯人、工程师、观测员使用统一人类骨架，便于后续复用待机、移动、受击等动画结构；影角色单独使用黑雾骨架和粒子层。
